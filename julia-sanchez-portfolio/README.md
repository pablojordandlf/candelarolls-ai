# Julia Sánchez Portfolio

Portfolio web profesional para Julia Sánchez, Directora de Marketing B2B SaaS.

## 🚀 Tecnologías

- **Next.js 15** - Framework React con App Router
- **TypeScript** - Tipado estático
- **Tailwind CSS** - Estilos utilitarios
- **Framer Motion** - Animaciones fluidas
- **Lucide React** - Iconos

## 🎨 Design System

### Colores
- Background: `#0A0A0B` (near-black)
- Secondary: `#141415` (elevated surfaces)
- Accent: `#F5F5F5` (primary text)
- Muted: `#A1A1AA` (secondary text)
- Border: `#52525B` (subtle borders)

### Tipografía
- **Primary**: Inter (sans-serif)
- **Monospace**: JetBrains Mono (números y métricas)

## 🛠️ Desarrollo

```bash
# Instalar dependencias
npm install

# Servidor de desarrollo
npm run dev

# Build de producción
npm run build
```

## 📦 Deploy

El proyecto está configurado para exportación estática:

```bash
npm run build
```

Los archivos se generan en la carpeta `dist/` listos para subir a Vercel, Netlify, o cualquier hosting estático.

### Vercel (recomendado)

1. Conecta tu repo en [vercel.com](https://vercel.com)
2. El build se ejecuta automáticamente
3. Configura tu dominio personalizado
